# Dashboard

Launch the interactive TUI dashboard by running:

```bash
node "$CLAUDE_PLUGIN_ROOT/bin/tui.js"
```

The dashboard shows:
- **Overview**: Today's token usage, cost, active sessions, cache hit rate
- **Sessions**: Session history with duration, tokens, cost, and context peak

Navigation: Press 1-5 to switch views, q to quit, r to refresh.
